☁️ Conectar Firebase (Sincronización)
📋 ¿Cómo obtener tu configuración?
1. Ve a console.firebase.google.com
2. Selecciona tu proyecto → ⚙️ Configuración → Tus aplicaciones
3. Crea una app Web (si no tienes) y copia el objeto firebaseConfig
4. Pega cada valor en los campos de abajo.
⚠️ Pega esta regla en Firestore → Reglas → Publicar:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; } } } Con esta regla, cualquier dispositivo que tenga la configuración de arriba se conecta automático, sin pedir clave — ideal para usar el sistema desde tu PC, laptop y celular sin fricción. A cambio, cualquier persona que reciba este archivo HTML también podría acceder a la base de datos, así que mantén el archivo privado (no lo compartas ni lo subas a una web pública).
🔒 ¿Prefieres pedir clave en cada dispositivo nuevo? (opcional, más seguro)
1. Ve a Authentication → Sign-in method y activa "Correo electrónico/Contraseña".
2. En Users, crea 1 usuario con correo y contraseña.
3. Cambia la regla de Firestore a allow read, write: if request.auth != null;
4. Llena los campos de abajo con ese correo/contraseña — si los dejas vacíos, el sistema se conecta directo sin pedir nada (configuración recomendada para uso personal en varios dispositivos).
